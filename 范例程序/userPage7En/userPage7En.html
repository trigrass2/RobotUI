 Description and requirements
  
Title : Teach pendant based user interface tools

General description :
  This library provides tools to assist in the implementation of a
  teach pendant based semi-graphical user interface in VAL3.
  This manager provides programs allowing to display, enter data or manage a list
  of data in the userPage () of the CS8. 

Hardware required :
  - Robot system    : any Staubli robot
  - Peripherals     : no peripherals required

VAL3 options :
  - VAL3 version    : 7.1.3
  - VAL3 licenses   : PaintiXen or VALPlast in demo mode
  - VAL3 add-ons    : none

Library version     : 7.1.2.0

Public variables    :
  - bool bShowDialog : Set to true during every displaying of a dialog window :
    insertDialog(), questionDialog(), timerDialog(). You can't run two dialogs
    window at the same time.
  - bool bUseEscape : set to true when you press ESC and false otherwise.
    This value is updating in several programs which use a user dialog.
  - bool bUseRefresh : Set to true after every displaying of a dialog window :
    insertDialog(), questionDialog(), setView(), timerDialog().
    When you use several tasks you must do a refresh.
  - num nBottomX, nBottomY, nTopX, nTopY : position of the list menu. (See setSizeList())
  - num nPage : Utilisation to know the current page which is open.
    You could use it when you have several pages.
  - num nView : Give the current position of the list cursor : if you use a list with a size
    of 10 elements the current position is between 0 to 10
    (different that the position of the selected Items)
  - string sItems : This array of string, contains the list of the current data.

Included libraries  : none

Miscellaneous       : ---

Keywords            : GUI, manager, interface, gestionnaire, user interface,
                    interface utilisateur, list, liste, human-machine, homme-machine
                    semi-graphical, semi-graphique

public changeValue(screen& x_scr, num& x_nValue[---], num& x_nMin, num& x_nMax, num& x_nX, num& x_nY, string& x_sItem[---]) 
  Description of program
    Get a numerical value on the x_nX column and x_nY line and test if
    the value is between x_nMin and x_nMax. If the two parameters x_nMin
    and x_nMax are equal there are no limits.
    If x_sItems is not Empty, the program displays the list of Items and
    you can change the selected Item : the index is returned in x_nValue.
  
  Input parameters

      role : min value limit
    num x_nMax :

    num x_nX :
      role : X cursor position for change the value
    num x_nY :
      role : Y cursor position for change the value
  
  Output paramaters
    num x_nValue :
      role : numerical value changed or index pointer of x_sItem
    string x_sItems
      role : array of Items
  
  Example
    call changeValue(scrMy,l_nValue,0,10,2,5,"") //The user can changed the numerical
                                             value of l_nValue. If the seized
                                             is not between 0 to 10 the user must
                                             changed the value.

private example() 
  Description of program
    This is an example for use this library.
    This example performs the following actions :
    - Run the fhmns program (converted of num seconds in string hour)
    - add, insert, delete, and select an Item with an updating of the menu.
    - change Menu (list of Items) with left and right keys
    - clear the menu
    - Run a simple list (no array) for just test the scrolling
  
    The sequence is :
    - Creation of 2 string arrays for the lists of items
    - Start main loop
    -   Creation of the menu with 2 lists
    -   Reading events from the keyboard
    -     case TAB           : update the width of the list (enlarge/reduce)
    -     case left or right : update the item's list and change the menu
    -     case Enter         : displays the Item selected
    -     case F1 or ESC     : quit the program
    -     case F2            : clear th list
    -     case F3            : Run function wich converted seconds
                                -> string Hours:Min:Sec
    -     case F4            : test editNumField() function
    -     case F5            : creation of an other list : simple list
    -     case F6            : delete an Item
    -     case F7            : add an item at the end of the list
    -     case F8            : insert an item at the current position
    -   Update the Item's list
    -   Refresh the screen
  
  History

    Modified : 08-08-11 - J.Cook     Changed $setTextMode to setTextMode
                                     And replaced literals ("", 0) with local variables
